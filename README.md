
  # Design do jogo RPG
Este é um pacote de código para Design do jogo RPG. O projeto original está disponível em https://www.figma.com/design/hoYpQAznpVhN2bqrH1mzSm/Design-do-jogo-RPG.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  
 ## 🛠️ Debug no Chrome
O projeto já tem configuração para debug. Abra o Chrome em http://localhost:5173 e use as DevTools (F12).

O servidor de desenvolvimento estará rodando com hot-reload automático. 🎮
